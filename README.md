## 상품

- 생성
    - 이름, 가격, 재고수량, 판매자(아이디), 생성시점, 수정시점
- 조회
    - 이름, 가격, 재고수량, 판매자(닉네임, 이메일) 생성시점, 수정시점
- 변경
    - 판매 중지 : 상태변경
        - 재고수량과 무관하게 판매자의 요청에 따라 일시 품절 상태로 조회되도록 상태값 변경
    - 재고수량변경
        - 판매자의 요청대로 재고수량으로 변경된다
        - 구매자의 주문 수량만큼 재고 수량이 감소된다
